config_manager

UAV配置管理工具

此模块提供了一组函数和命令行接口，用于管理UAV模块的配置文件。 它允许用户创建、重置、更新和查看配置。

RS.UAV.config.config_manager.create_default_config()[源代码]

创建默认配置文件

将当前配置保存为默认配置文件，以便在需要时重置配置。

返回:

是否成功创建默认配置文件

返回类型:

bool

RS.UAV.config.config_manager.reset_config()[源代码]

重置配置为默认值

从默认配置文件加载配置并保存到主配置文件。

返回:

是否成功重置配置

返回类型:

bool

RS.UAV.config.config_manager.update_config(key_path, value)[源代码]

更新配置中的特定值

参数:

  • key_path (str) – 键路径，如 “paths.base_dir”

  • value (Any) – 新值

返回:

是否成功更新配置

返回类型:

bool

示例

>>> update_config("data_params.patch_size", "128")

True
